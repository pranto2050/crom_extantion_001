# LumiList - Personal Bookmark Manager

LumiList is a powerful and beautiful bookmark manager designed to help you organize your digital life with ease. It features a column-based board layout, password-protected pages, and customizable themes.

## 🚀 How to Use

### 1. Organizing Your Workspace
- **Pages**: Use the tabs at the top to create different workspaces (e.g., "Work", "Personal", "Research").
- **Boards**: Each page contains 4 columns where you can add "Boards" to group related links.
- **Bookmarks**: Click the `+` button inside any board to add a new bookmark. You can fetch titles automatically by entering a URL.

### 2. Customization
- **Drag and Drop**: You can drag boards between columns or reorder bookmarks within a board.
- **Compact Mode**: Enable Compact Mode in Settings to see more bookmarks at once.
- **Font Colors**: Change the text color of your boards and bookmarks in Settings.

### 3. Searching
- Click the **Search** icon (magnifying glass) in the bottom-right rail to search across all your boards, titles, and descriptions.

## 🖼️ How to Change the Wallpaper

1. Look for the **Style and wallpaper** icon (the palette icon) in the floating controls rail at the bottom-right of your screen.
2. Click the icon to open the Wallpaper Sidebar.
3. Browse through the available categories (Dark, Light).
4. Click on any image to apply it as your workspace background.
5. Your choice will be saved automatically for the current theme mode.

## 🔒 Security & Passwords

LumiList provides two levels of security: **Admin Password** and **Page Passwords**.

### 1. Admin Password
The Admin Password protects the **Security** tab in the Settings menu, preventing unauthorized users from locking/unlocking pages.

- **Default Admin Passwords**:
  - `2050` (Kaji)
  - `1234` (Rahim)
  - `5678` (Karim)
  - `1234` (Kardim)

- **How to Change the Admin Password**:
  Since this is a browser extension, the admin credentials are stored in the project files for security. To change them:
  1. Navigate to the extension folder.
  2. Open `newtab/admin/admin.json`.
  3. Edit the `password` field for the desired user.
  4. Save the file and reload the extension.

### 2. Page Passwords (Locking Pages)
You can restrict access to specific pages by setting a password.

- **How to Set a Page Password**:
  1. Click the **Settings** icon (gear icon) in the bottom-right rail.
  2. Click the **Security** tab.
  3. Enter the **Admin Password** when prompted.
  4. Under "Lock Page", select the page you want to protect from the dropdown.
  5. Enter a password for that page and click **Lock Selected Page**.
- **How to Use**: When you try to switch to a locked page, LumiList will prompt you for the page's password before revealing its contents.
- **How to Change/Remove**: In the **Security** tab (after Admin authentication), you will see a list of "Locked Pages" where you can manage or unlock them.

## 🛠️ Advanced Settings

Access the **Settings** menu to toggle additional features:
- **Privacy Blur**: Quickly blur all bookmark titles for privacy.
- **Incognito Mode**: Toggle whether links should open in a new Incognito window.
- **Usage Tracking**: Enable insights to see which bookmarks you use most frequently.
- **Clock**: Toggle the live clock display.
